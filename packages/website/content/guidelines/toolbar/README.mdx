# Toolbar

## Purpose

Provides persistent, in-context access to complementary tools so users can stay focused on their primary editing task.

## When to Use

Describe the user goals, scenarios, and constraints that make this pattern appropriate.

### Typical use cases

- Accessing AI agents while staying in the current workflow
- Managing tags, taxonomies and entity metadata 
- Using analytics or performance insights in-context while editing or reviewing an Experience
- Accessing personalization controls such as variants or audience-specific configuration
- Triggering workflows and automations alongside the main task
- Surfacing apps and third-party integrations relevant to the current page or editor context
