Flex components is [Box] with `display: flex` and abilities to apply other flexbox CSS properties via properties of the component.

The Box component is useful because it helps with several common cases:

- Create a layout with uses Forma36 spacing
- Compose new component and allow to override element using the `as` property.

## Import

```js static=true
import { Flex } from '@contentful/f36-components';
```

## Usage

### Margins & Paddings

Flex components inherits behaviour of [Box] component and allows to pass margins and paddings that are based on our [spacing system].

## Usage

```js
() => {
  const ColorBox = ({ backgroundColor, ...rest }) => (
    <Flex
      as="div"
      justifyContent="center"
      alignItems="center"
      style={{
        width: '120px',
        height: '120px',
        color: tokens.colorWhite,
        backgroundColor,
      }}
      {...rest}
    />
  );

  return (
    <Flex
      justifyContent="space-between"
      alignItems="center"
      marginBottom="spacingM"
    >
      <ColorBox backgroundColor={tokens.colorGreenBase}>Example item</ColorBox>
      <ColorBox backgroundColor={tokens.colorRedBase}>Example item</ColorBox>
      <ColorBox backgroundColor={tokens.colorBlueBase}>Example item</ColorBox>
    </Flex>
  );
};
```

<br />

```jsx
<Flex justifyContent="space-between" alignItems="center">
  <Box>
    <Paragraph>User name</Paragraph>
  </Box>
  <Flex alignItems="center">
    <Box marginRight="spacingM">
      <TextLink href="http://google.com">User profile</TextLink>
    </Box>
    <Box>
      <Button>Add user to the team</Button>
    </Box>
  </Flex>
</Flex>
```
