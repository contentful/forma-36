Box is the most abstract layout component that allows to apply `margin` and `padding` properties according to [Forma36 spacing](../../foundation/spacing). By default, it renders a `div` element.

The Box component is useful because it helps with several common cases:

- Create a layout with uses Forma36 spacing
- Compose new component and allow to override element using the `as` property.

## Import

```js static=true
import { Box } from '@contentful/f36-components';
```

## Usage

```jsx
() => {
  const SmallWhiteBox = () => (
    <Box
      as="span"
      style={{
        width: '30px',
        height: '30px',
        backgroundColor: tokens.colorWhite,
      }}
    />
  );

  return (
    <div style={{ display: 'flex', alignItems: 'center' }}>
      <Box
        padding="spacingXs"
        marginRight="spacingM"
        style={{
          backgroundColor: tokens.colorRedBase,
        }}
      >
        <SmallWhiteBox />
      </Box>
      <Box
        padding="spacingXs"
        marginRight="spacingM"
        style={{
          backgroundColor: tokens.colorBlueBase,
        }}
      >
        <SmallWhiteBox />
      </Box>
    </div>
  );
};
```

### as property

You can use the `as` prop to change the element render, just like styled-components.

```jsx
<Box as="button" padding="spacingXs" margin="spacingS">
  Button
</Box>
```
