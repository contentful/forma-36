Select Field is used for choosing a single option from a list of options.

```jsx
<React.Fragment>
  <SelectField labelText={'Label'}>
    <Option value="optionOne">Option 1</Option>
    <Option value="optionTwo">Option 2</Option>
    <Option value="optionThree">Option 3</Option>
    <Option value="optionFour">Option 4</Option>
  </SelectField>
</React.Fragment>
```

## Best practices <a name="best-practices" />

- Add a label that is clear what the user is selecting.
- Use when there is 4 or more options presented. Use RadioButtonField for 3 options or less.
- Whereever possible choose one of the options as a default selection.

## Variations <a name="variations" />

The select field follows standard field variations, as follows:

### Select Field (`selectFieldTypes`) <a name="selectFieldTypes-types" />

```jsx
<React.Fragment>
  <SelectField labelText={'Label'}>
    <Option value="optionOne">Option 1</Option>
    <Option value="optionTwo">Option 2</Option>
    <Option value="optionThree">Option 3</Option>
    <Option value="optionFour">Option 4</Option>
  </SelectField>
</React.Fragment>
```

### Required <a name="selectFieldTypes-required"/>

```jsx
<React.Fragment>
  <SelectField labelText="Required" required>
    <Option value="optionOne">Option 1</Option>
    <Option value="optionTwo">Option 2</Option>
    <Option value="optionThree">Option 3</Option>
    <Option value="optionFour">Option 4</Option>
  </SelectField>
</React.Fragment>
```

### Disabled <a name="selectFieldTypes-disabled"/>

```jsx
<React.Fragment>
  <SelectField labelText="isDisabled" selectProps={{ isDisabled: true }}>
    <Option value="optionOne">Option 1</Option>
    <Option value="optionTwo">Option 2</Option>
    <Option value="optionThree">Option 3</Option>
    <Option value="optionFour">Option 4</Option>
  </SelectField>
</React.Fragment>
```

### Help text <a name="selectFieldTypes-helpText"/>

```jsx
<React.Fragment>
  <SelectField labelText="With Help Text" helpText="This is some Help Text">
    <Option value="optionOne">Option 1</Option>
    <Option value="optionTwo">Option 2</Option>
    <Option value="optionThree">Option 3</Option>
    <Option value="optionFour">Option 4</Option>
  </SelectField>
</React.Fragment>
```

### Validation Message <a name="selectFieldTypes-validationMessage"/>

```jsx
<React.Fragment>
  <SelectField
    labelText="Validation Message"
    validationMessage="This is a validation message"
  >
    <Option value="optionOne">Option 1</Option>
    <Option value="optionTwo">Option 2</Option>
    <Option value="optionThree">Option 3</Option>
    <Option value="optionFour">Option 4</Option>

```

## Copy considerations <a name="copy-considerations" />

- If there is no clear default option the default text should be Select.
- Try to make the copy for the options as direct as possible. ie "Publish"
