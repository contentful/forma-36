The Form component is a wrapper component to apply a consistent layout and handle form submissions.

```jsx
<React.Fragment>
  <Form onSubmit={() => console.log('onSubmit')} spacing="default">
    <FieldGroup>
      <TextField
        required
        name="nameInput"
        id="nameInput"
        labelText="Name"
        value="Johannes Bugiel"
        helpText="Enter name"
      />
      <TextField
        required
        name="emailInput"
        id="emailInput"
        labelText="E-Mail"
        value="johannes.bugiel@contentful.com"
        helpText="Enter email address"
      />
    </FieldGroup>
    <FieldGroup>
      <Button>Submit</Button>
    </FieldGroup>
  </Form>
</React.Fragment>
```

## Best practices <a name="best-practices" />

- The `Form` component should wrap FieldGroup components to apply consistent layouts
- Provide an `onSubmit` handler for form validation and submission
