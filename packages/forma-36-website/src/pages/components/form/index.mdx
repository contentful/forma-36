Form is a wrapper component that handles form submissions.

```jsx
<React.Fragment>
  <Form
      onSubmit={()=> console.log('onSubmit')}
      spacing="default"
    >
      <TextField
        required
        name="nameInput"
        id="nameInput"
        labelText="Name"
        value="Johannes Bugiel"
        helpText="Please enter your name"
      />
      <TextField
        required
        name="emailInput"
        id="emailInput"
        labelText="E-Mail"
        value="johannes.bugiel@contentful.com"
        helpText="Please enter your mail"
      />
      <Button>Submit</Button>
    </Form>
</React.Fragment>
```

## Best practices <a name="best-practices" />

- This `Form` component wraps it's contents with HTML `<form>` component
- Should be used to wrap all form input components
- To imitate navive HTML behaviuor, add handler for `oncSubmit` action
