`TextField` is a composition of the `FormLabel`, `TextInput`, `HelpText` and `ValidationMessage` components.
It allows users to input text or numbers and should be used in a form context.

```jsx
<React.Fragment>
  <TextField labelText="Email address" helpText="Please enter your email address" required />
</React.Fragment>
```

## Best practices <a name="best-practices" />

- To ensure `TextField` meets accessibility standards, provide `labelText`, `name` and `id`

## TextField variations <a name="variations" />

The TextField component can be configured in a number of different ways. Use this guidance to determine the correct variation based on the context. 

### Compositional properties <a name="compositional-properties" />

Since the TextField is a composition of multiple components, it is possible to pass child component specific properties. 
For this purpose, the TextField exposes the following properties:

```jsx
<React.Fragment>
  <TextField
    labelText="E-mail address"
    helpText="Please enter your email address"
    textInputProps={{ withCopyButton: true }}
    textLinkProps={{ text: 'privacy' }}
    formLabelProps={{ requiredText: 'required' }}
    required
  />
</React.Fragment>
```

| Property           | Description                                               |
| :----------------- | :-------------------------------------------------------- |
| **textInputProps** | Used to pass a property object to the TextInput component |
| **formLabelProps** | Used to pass a property object to the FormLabel component |
| **textLinkProps**  | Used to pass a property object to the TextLink component  |

### Label and text properties (`labelText, helpText, validationMessage`) <a name="label-and-text" />

```jsx
<React.Fragment>
  <TextField
    labelText="E-mail address"
    helpText="Please enter your email address"
    validationMessage="This address is not valid"
  />
</React.Fragment>
```

| Property              | Description                                                   |
| :-------------------- | :------------------------------------------------------------ |
| **labelText**         | Used to provide the text                                      |
| **helpText**          | Used for entering passwords or sensitive information          |
| **validationMessage** | Used for E-mail address addresses (will dispatch native autocomplete) |

### Required field (`required`) <a name="required-field" />

```jsx
<React.Fragment>
  <TextField labelText="E-mail address" helpText="Please enter your email address" required />
</React.Fragment>
```

| Property     | Description                                                                    |
| :----------- | :----------------------------------------------------------------------------- |
| **required** | Used to set the input as required and show the required text next to the label |

### Render textarea (`textfield`) <a name="render-textarea" />

Instead of an input, the TextField component can render a text area for long text requirements.

```jsx
<React.Fragment>
  <TextField
    labelText="Description"
    helpText="Please enter a description"
    textarea
  />
</React.Fragment>
```

| Property     | Description                                      |
| :----------- | :----------------------------------------------- |
| **textarea** | Used to render a textarea instead of a textinput |

### Additional TextLink (`textLinkProps`) <a name="additional-textlink" />

The TextField can be configured to display an additional TextLink in the label.

```jsx
<React.Fragment>
  <TextField
    labelText="User Name"
    textLinkProps={{
      icon: 'Lock',
      text: 'Unlock to edit',
      onClick: e => {
        console.log('textLinkClick', e);
      },
    }}
    value="Lorem Ipsum"
    textInputProps={{
      disabled: true,
    }}
  />
</React.Fragment>
```

| Property                  | Description                                           |
| :------------------------ | :---------------------------------------------------- |
| **textLinkProps**         | Property object used to define the rendered text link |
| **textLinkProps.icon**    | Used to specify the rendered icon                     |
| **textLinkProps.text**    | Used to specify the link text                         |
| **textLinkProps.onClick** | Used to specify an onClick handler                    |

### Character count (`countCharacters`) <a name="character-count" />

The TextField can be configured to display a character count. This is useful when an input field has a character limit.<br/>
**Note**: The `maxLength` property must be provided in the TextInput component using `textInputProps`.

```jsx
<React.Fragment>
  <TextField
    labelText="User Name"
    countCharacters
    value="Lorem Ipsum"
    textInputProps={{ maxLength: 20 }}
  />
</React.Fragment>
```

| Property            | Description                        |
| :------------------ | :--------------------------------- |
| **countCharacters** | Used to enable the character count |

## Writing guidelines <a name="writing-guidelines" />

- Use direct, succinct copy that helps the user to successfully complete the form
- Do not preface the instructions with introductory text, such as "please"
- To add additional context, link out to documentation
- In cases where additional context is unnecessary, help text can be substituted for text in the label 
- Do not use punctuation for labels, help text and links
- Use sentence style caps (in which only the first word of a sentence or term is capitalized)
