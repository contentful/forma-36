FormLabels make it possible to define input label components.

```jsx
<React.Fragment>
  <FormLabel htmlFor="name">
    Full name
  </FormLabel>
  <TextInput
    name="name"
    value="Claus Mitchell"
    className="f36-margin-bottom--xs"
  />
</React.Fragment>
```

## Best practices

- Provide html for all properties of the input label

## Copy considerations

- Do not use punctuation following a label
- Use sentence style caps (in which only the first word of a sentence or term is capitalized) on labels

## FormLabel required properties (`required, requiredText`)

To communicate that the corresponding input of the Formlabel is required, use the props `required` and `requiredText`. 
Use this guidance to determine the correct required properties to use based on the context. 

```jsx
<React.Fragment>
  <FormLabel htmlFor="example1" required>
    Name
  </FormLabel>
  <TextInput
    name="example1"
    value="Claus Mitchell"
    className="f36-margin-bottom--m"
  />
  <FormLabel htmlFor="example2" required requiredText="required">
    Email address
  </FormLabel>
  <TextInput value="claus.mitchell@contentful.com" name="example2" />
</React.Fragment>
```

| Property         | Description                                       |
| :--------------- | :------------------------------------------------ |
| **required**     | Used to add the text 'required' next to the label |
| **requiredText** | Used to customize the 'required' text             |
