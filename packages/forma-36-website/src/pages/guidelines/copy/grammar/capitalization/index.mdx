# Capitalization

## Use sentence-style caps
Sentence style caps, (capitalizing only the first word of a sentence) should be used for:
  - labels
  - headings
  - titles
  - menu items

![Build Status](/sentence-style-caps.png)

## Contentful UI elements
Contentful UI elements should be written in lowercase unless the term is proprietary.
For example:  
  - space
  - entry
  - content type
