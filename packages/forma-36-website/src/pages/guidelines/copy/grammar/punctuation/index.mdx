# Punctuation

## Use periods consistently

Keep UX copy scannable by eliminating unnecessary periods.
Do not use periods on:

- buttons
- labels
- tooltips
- bulleted lists

Use periods on:

- multiple sentences
- any sentence followed by a link
- modal body text

## Avoid exclamation marks

Exclamation marks may come across as shouting and do not match the brand tone established for the web app. There are some exceptions when it is appropriate to use exclamation marks, such as a greeting or congratulatory message.

![Avoid exclamation marks](/exclamation-mark.jpg)

## Use colons consistently

Colons are useful when introducing text or giving emphasis, but should be avoided in stand-alone copy.
Do not use colons on:

- labels

Use colons on:

- introducing lists or text
