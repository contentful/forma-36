# Component examples

## Modal example
The purpose of modals are to communicate actions to a user while they are navigating. The copy should be clear, direct and help a user to quickly assess their options.

![Modal example](/modal-example.png)

## Modal copy checklist
1.  Are your sentences and labels short, with simple language?
2.	Did you include as few concepts as possible?
3.	Did you use present tense?
4.	Did you use sentence style capitalization?
5.	Is your copy positive, inclusive and inviting? 	
6.	Did you begin sentences with an objective wherever possible?
