## How to use BaseCheckbox

- You only need to define a suitable `label` to get a basic controlled input.
- Create a checkbox, switch or radio button by defining the `type`.
- All common states (disabled, required, checked, indeterminate) are functionally and visually supported.
- Listen to native element events via `onChange`, `onBlur`, `onFocus`, `onKeyDown`. More event handlers can be attached by defining them via `inputProps`.

### Using as a controlled input

You can use the BaseCheckbox as a controlled input, for that you need to:

- Pass the `isChecked` property, with this property it will already be a controlled component;
- Pass a `onChange` handler, so you can use it to update the value of `isChecked`;

Setting the `isChecked` will already define it as a controlled input.

```jsx static=true
const ExampleControlled = () => {
  const [checkboxState, setCheckboxState] = useState(false);

  return (
    <BaseCheckbox
      type="checkbox"
      name="newsletter-subscribe-controlled"
      id="newsletter-subscribe-controlled"
      isChecked={checkboxState}
      onChange={() => setCheckboxState(!checkboxState)}
    >
      Subscribe to newsletter
    </BaseCheckbox>
  );
};
```

### Using as an uncontrolled input

You can use the BaseCheckbox as an uncontrolled input, for that you can:

- Set the `defaultChecked` property, it will ensure that the checked state can be altered normaly.
- Don't set the `isChecked` as it will make the input controlled.

```jsx
<BaseCheckbox
  type="checkbox"
  name="newsletter-subscribe-uncontrolled"
  id="newsletter-subscribe-uncontrolled"
  defaultChecked={true}
>
  Subscribe to newsletter
</BaseCheckbox>
```

## Component variations

- **Checkbox** - render a checkbox field.

```jsx
<BaseCheckbox type="checkbox" name="newsletter-subscribe">
  Subscribe to newsletter
</BaseCheckbox>
```

- **Radio button** - render a radio button field.

```jsx static=true
function RadioButtonExample() {
  const [activeOption, setActiveOption] = React.useState('');
  return (
    <Flex flexDirection="column">
      <BaseCheckbox
        id="pill-decision-1"
        type="radio"
        name="pill-decision"
        value="blue"
        isChecked={activeOption === 'blue'}
        onChange={(e) => {
          setActiveOption(e.target.value);
        }}
      >
        Take the blue pill
      </BaseCheckbox>
      <BaseCheckbox
        id="pill-decision-2"
        type="radio"
        name="pill-decision"
        value="red"
        isChecked={activeOption === 'red'}
        onChange={(e) => {
          setActiveOption(e.target.value);
        }}
      >
        Take the red pill
      </BaseCheckbox>
    </Flex>
  );
}
```

- **Switch** - render a switch field.

```jsx
<BaseCheckbox id="switch" type="switch" name="notification">
  Allow notifications
</BaseCheckbox>
```
