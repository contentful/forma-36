Use SwitchField to show the users a switch toggle. It contains a checkbox input with its label. If needed the component can display an error message and hint message. For a multiple-choice, please consider using [CheckboxField component](https://f36.contentful.com/components/checkbox-field/), or the [RadioButttonField component](https://f36.contentful.com/components/radion-button-field/) for a single choice.

## How to use SwitchField

- Switch should work independently, checking one should not affect any other checkbox in the interface
- Switch should always have a label
- Be explicit about the action that will follow if the switch is active

## Component variations

- **Basic usage**

```jsx
<SwitchField id="randomId" label="Label text" />
```

- **With helper text** - Example with helper text displayed

```jsx
<SwitchField
  id="my-switch"
  label="Some label text"
  name="my-switch"
  helpText="Some help text"
/>
```

## Content recommendations

- Labels should be concise and provide context.
- Use sentence-style capitalization (only the first word in a phrase and any proper nouns capitalized)
