Use the SkeletonDisplayText component to simiulate headings, subheadings or titles whenever loading asynchronous data.

```jsx
<SkeletonContainer>
  <SkeletonDisplayText />
</SkeletonContainer>
```

## Best practices <a name="best-practices" />

- Each SkeletonDisplayText needs to be wrapped by a SkeletonContainer component
- Only use it to mimic headings and titles, for normal text we recommend using SkeletonBodyText.
