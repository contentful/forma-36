The `@contentful/f36-datetime` package provides a couple of functions to format a date.

## Import

```jsx static=true
import {
  formatDateAndTime,
  formatMachineReadableDateTime,
  formatRelativeDateTime,
} from '@contentful/f36-components';
// or
import {
  formatDateAndTime,
  formatMachineReadableDateTime,
  formatRelativeDateTime,
} from '@contentful/f36-datetime';
```

## formatDateAndTime

This function will return a formatted string of any date that is passed as the first argument of the function.
It allows 4 different formats that can be chosen by passing one of the `format` options as a second argument of the function:

<Table>
  <Table.Head>
    <Table.Row>
      <Table.Cell>
        <Subheading>`format` option</Subheading>
      </Table.Cell>
      <Table.Cell>
        <Subheading>Result</Subheading>
      </Table.Cell>
    </Table.Row>
  </Table.Head>
  <Table.Body>
    <Table.Row>
      <Table.Cell>`full`</Table.Cell>
      <Table.Cell>12 Aug 2020 at 8:00 AM</Table.Cell>
    </Table.Row>
    <Table.Row>
      <Table.Cell>`fullWithSeconds`</Table.Cell>
      <Table.Cell>12 Aug 2020 at 8:00:00 AM</Table.Cell>
    </Table.Row>
    <Table.Row>
      <Table.Cell>`day`</Table.Cell>
      <Table.Cell>12 Aug 2020</Table.Cell>
    </Table.Row>
    <Table.Row>
      <Table.Cell>`time`</Table.Cell>
      <Table.Cell>8:00 AM</Table.Cell>
    </Table.Row>
    <Table.Row>
      <Table.Cell>`weekday`</Table.Cell>
      <Table.Cell>Wed, 12 Aug</Table.Cell>
    </Table.Row>
  </Table.Body>
</Table>

If no format is passed to the function, it will return a string with the `full` format

## formatMachineReadableDateTime

This function will return a machine-readable date string that should be passed to the `datetime` attribute of a `<time>` tag.
It allows four different formats that can be chosen by passing one of the `format` options as a second argument to the function:

<Table>
  <Table.Head>
    <Table.Row>
      <Table.Cell>
        <Subheading>`format` option</Subheading>
      </Table.Cell>
      <Table.Cell>
        <Subheading>Result</Subheading>
      </Table.Cell>
    </Table.Row>
  </Table.Head>
  <Table.Body>
    <Table.Row>
      <Table.Cell>`full`</Table.Cell>
      <Table.Cell>2019-08-24T15:44:00.000Z</Table.Cell>
    </Table.Row>
    <Table.Row>
      <Table.Cell>`day`</Table.Cell>
      <Table.Cell>2019-08-24</Table.Cell>
    </Table.Row>
    <Table.Row>
      <Table.Cell>`time`</Table.Cell>
      <Table.Cell>15:44:07.000</Table.Cell>
    </Table.Row>
    <Table.Row>
      <Table.Cell>`weekday`</Table.Cell>
      <Table.Cell>08-24</Table.Cell>
    </Table.Row>
  </Table.Body>
</Table>

If no format is passed to the function, it will return a string with the `full` format

## formatRelativeDateTime

This function will return a relative date string such as "in a day", "in one month", or "one month ago" when comparing the given date to "now" or to the optional base date.
