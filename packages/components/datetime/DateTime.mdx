In `@contentful/f36-datetime`, there are two components (`DateTime` and `RelativeDateTime`) that will return a date following [Forma 36 guidelines](https://f36.contentful.com/guidelines/date-and-time/).

## DateTime

The DateTime component will format any date to a human friendly format and wrap it in a `<time>` tag.
It allows four different forms of display that can be controlled by changing the `format` prop:

| `format` option | Result                 |
| --------------- | ---------------------- |
| `full`          | 12 Aug 2020 at 8:00 AM |
| `day`           | 12 Aug 2020            |
| `time`          | 8:00 AM                |
| `weekday`       | Wed, 12 Aug            |

### Examples

```tsx
<Grid rowGap="spacingX" rows="repeat(3, auto)">
  {/* passing a Timestamp as an ISOString */}
  <DateTime date="2021-08-17T15:45" />
  {/* passing a JS Date */}
  <DateTime date={new Date()} />
  {/* passing a Unix Epoch in milliseconds */}
  <DateTime date={1629240300000} />
</Grid>
```

All the usages above will result in 'Tue, 17 Aug 2021 at 3:45 PM'
