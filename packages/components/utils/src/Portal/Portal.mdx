`Portal` component can be used to render children in to a DOM node outside of the parent component. It can be useful for rendering components, you dont want to be hidden, clipped by parent component. For example, models, popovers, dropdowns.

## Import

```jsx static=true
import { Portal } from '@contentful/f36-utils';
```

## Example

### Basic usage

```jsx static=true
<Box>
  <Portal>
    <Box
      style={{
        backgroundColor: 'gray',
      }}
    >
      Hi! I'm first in the document flow, but will be rendered at the end of the
      document!
    </Box>
  </Portal>
  Parent component content
</Box>
```
