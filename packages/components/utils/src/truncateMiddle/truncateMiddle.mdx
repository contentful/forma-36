`truncateMiddle` is a utility function that truncates a given string from the middle and replaces the removed characters with a customizable string.

## Import

```jsx static=true
import { truncateMiddle } from '@contentful/f36-utils';
```

## Example

### Basic usage

```ts static=true
import { truncateMiddle } from '@contentful/f36-utils';

truncateMiddle('long string', { start: 3, end: 2 });
// lonâ€¦ng

truncateMiddle('long string', { start: 3, replacement: '***' });
// lon***
```
