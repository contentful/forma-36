`truncateMiddle` is a utility function that truncates a given string to a specific length and replaces the removed characters with a customizable string.

## Import

```jsx static=true
import { truncateMiddle } from '@contentful/f36-utils';
```

## Example

### Basic usage

```ts static=true
import { truncateMiddle } from '@contentful/f36-utils';

truncateMiddle('long string', 5);
// loâ€¦ng

truncateMiddle('long string', 7, '***');
// lo***ng
```
