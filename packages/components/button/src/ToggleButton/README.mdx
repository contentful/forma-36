Example of simple usage:

```jsx

// import { PreviewIcon } from '@contentful/f36-icons';
```

Example of active and disabled toggle buttons:

```jsx

<>

  </ToggleButton>
  <br />
  <ToggleButton testId="toggle-button" isDisabled onToggle={() => {}}>
    Disabled Toggle Button
  </ToggleButton>
</>
```
