`Caption` is used to display secondary and tertiary information, and in special scenarios where information should be de-emphasized.

## Import

```js static=true
import { Caption } from '@contentful/f36-components';
// or
import { Caption } from '@contentful/f36-typography';
```

## Density support

This component supports multiple densities thanks to the [useDensity](/hooks/use-density) hook and automatically adjusts its styling for each density (when wrapped with the `DensityProvider`).
