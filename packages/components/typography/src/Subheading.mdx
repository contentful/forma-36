`Subheading` is intended to be used as a heading for copy nested within a section. EX: a subheading under Contact may be Address or Phone number.

### Import

```js static=true
import { Subheading } from '@contentful/f36-components';
// or
import { Subheading } from '@contentful/f36-typography';
```

## Code examples

```jsx
<Subheading>I love Forma 36 design system.</Subheading>
```

You can override the default margin bottom by specifying it explicitly.

```jsx
<Flex flexDirection="column">
  <Subheading marginBottom="none">I love Forma 36 design system.</Subheading>
  <Subheading marginBottom="spacingXl">
    I love Forma 36 design system.
  </Subheading>
  <Subheading marginTop="spacingS">I love Forma 36 design system.</Subheading>
</Flex>
```
