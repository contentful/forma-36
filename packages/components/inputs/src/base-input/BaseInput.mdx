BaseInput is used to controll inputs, like TextInput, Textarea, or Select. We don't recommend using this component, it's considered an internal component that drives Contentful form components.
In order to use form elements, take a look at the documentation of:

- TextInput
- Textarea
- Select
- checkbox
- RadioButton

Note: You need to create your custom label or do your design require input without a label? Each of the components provide an `isStandalone` property, that allow you to use a specific component without a visible label (it will be added as an aria-label, so your input passes the a11y requirements).
