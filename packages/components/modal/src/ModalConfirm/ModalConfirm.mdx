`ModalConfirm` is a convenience wrapper on top of `Modal`. It is intended to use for showing confirmation dialogs.

### Import

```js static=true
import { ModalConfirm } from '@contentful/f36-components';
// or
import { ModalConfirm } from '@contentful/f36-modal';
```

## Code examples

```jsx
function ModalConfirmExample() {
  const [isShown, setShown] = React.useState(false);
  return (
    <div>
      <Button variant="negative" onClick={() => setShown(true)}>
        Delete something
      </Button>
      <ModalConfirm
        isShown={isShown}
        onCancel={() => {
          setShown(false);
        }}
        onConfirm={() => {
          setShown(false);
        }}
        onSecondary={() => {
          setShown(false);
        }}
      >
        <Text>Do you want to confirm your action?</Text>
      </ModalConfirm>
    </div>
  );
}
```
