## Code examples

Example of simple usage:

```jsx
// import { ToggleButton } from '@contentful/f36-togglebutton';

<ToggleButton testId="toggle-button" onToggle={() => {}} icon={Preview}>
  Toggle Button
</ToggleButton>
```

Example of active and disabled toggle buttons:

```jsx
// import { ToggleButton } from '@contentful/f36-togglebutton';

<>
  <ToggleButton testId="toggle-button" isActive onToggle={() => {}}>
    Active Toggle Button
  </ToggleButton>

  <ToggleButton testId="toggle-button" isDisabled onToggle={() => {}}>
    Disabled Toggle Button
  </ToggleButton>
</>
```
