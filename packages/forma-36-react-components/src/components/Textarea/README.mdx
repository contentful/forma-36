`Textarea` is a form element that allows the user to enter a sizeable amount of multi-line plain text.

## Table of contents

- [How to use Textarea](#how-to-use-Textarea)
- [Code examples](#code-examples)
- [Accessibility](#accessibility)

## How to use Textarea

The Textarea component can be configured in a number of different ways. Use this guidance to determine the best configuration based on the context.

## Code examples

```tsx
// import { Textarea } from '@contentful/forma-36-react-components';

<Textarea
  name="textarea"
  id="textarea"
  width="large"
  value="Example text in the textarea"
  rows={2}
/>
```

### Textarea state (`disabled`)

```tsx
<Textarea name="textarea" id="textarea" disabled value="123456" rows={2} />
```

### Textarea can be controlled by passing a `textareaRef` prop

```tsx
<Textarea
  name="textarea"
  id="textarea"
  disabled
  value="123456"
  rows={2}
  textareaRef={}
/>
```

## Accessibility

- To ensure accessibility, provide a `name` prop
