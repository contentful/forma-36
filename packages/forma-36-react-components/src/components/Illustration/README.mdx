Illustration is a component that helps convey complex ideas in a graphic way.

## Table of contents

- [How to use Illustration](#how-to-use-empty-state)
- [Code examples](#code-examples)

## How to use Illustration

Use Illustration when your application needs a graphic enhancement / aid.
It can be used as a standalone component or alongside text.

## Code examples

```jsx
<>
  <Illustration illustration="Archive" />
</>
```
